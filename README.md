# The Mind's Eye · 心灵之眼

**Reality Before Reality · 现实诞生之前** — an interactive, bilingual (English + 简体中文) exploration of imagination, mental simulation, spatial intelligence, and invention. Every machine, city, and invention existed first inside a human mind; this is the museum of that hidden faculty.

A da-Vinci-codex × engineering-blueprint experience: a cinematic hero where a floating wireframe **machine morphs into neurons, then into a mind**, a hands-on **Mental Rotation Lab** (real Shepard–Metzler "same or mirror?" 3D trials with live scoring), a gallery of **ten minds that saw first** (the Inventor, da Vinci, Tesla, the Engineer, Architect, Scientist/Einstein, Chess Master, Programmer, Urban Planner, the Future Machine — each with its own visualization), the **neuroscience of imagination**, the **spectrum of spatial ability**, the **Invention Engine** (Imagination → Concept → Sketch → Model → Prototype → Product → Civilization), the **history of human imagination**, an interactive **Imagination Knowledge Graph**, and a curated **AI Mind Guide**.

## Stack

Single self-contained `index.html` — vanilla JS + HTML5 Canvas (including a tiny 3D wireframe engine for the rotation lab), no build step, no dependencies. Fonts: Space Grotesk / Sora / Newsreader / JetBrains Mono / Noto Serif SC / Noto Sans SC. Honors `prefers-reduced-motion`.
